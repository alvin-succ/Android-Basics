# Android第一行代码 - 现代化改造

在学习Android第一行代码中，许多用法已过时，本项目主要针对这些Demo进行了修改。包括几个简单易上手的安卓Demo，帮助快速掌握安卓基础知识。

---

## 📱 主要功能演示

### 1. Activity之间传递数据
- **演示项目**：ActivityTest
- **主要功能**：演示如何向下个Activity传递数据，以及如何返回数据给前一个Activity
- **技术要点**：Intent数据传递、ActivityResultLauncher、Bundle使用
- **文章链接**：[Activity之间传递数据完整教程](https://blog.csdn.net/weixin_45769562/article/details/156360677?spm=1001.2014.3001.5502)

### 2. 相册相机调用
- **演示项目**：CameraAlbumTest
- **主要功能**：包括如何调用系统相机、相册，以及如何分享到微信及其他应用
- **技术要点**：相机权限、图片选择、Intent分享、文件存储
- **文章链接**：[相册相机调用完整教程](https://blog.csdn.net/weixin_45769562/article/details/155890289?spm=1001.2014.3001.5502)

### 3. Service使用示例
- **演示项目**：ServiceTest
- **主要功能**：演示Service的基本用法和生命周期管理
- **技术要点**：Service启动方式、绑定服务、前台服务、跨进程通信
- **文章链接**：[Service使用示例教程](https://blog.csdn.net/weixin_45769562/article/details/155890289?spm=1001.2014.3001.5502)
