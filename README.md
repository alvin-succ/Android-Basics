在学习Android第一行代码第二版中，许多用法已过时，本项目主要针对这些Demo进行了修改。包括几个简单易上手的安卓Demo,帮助快速掌握安卓基础知识。

主要包含以下几个Demo：

1. [Activity之间传递数据](https://blog.csdn.net/weixin_45769562/article/details/156360677?spm=1001.2014.3001.5502)
   ActivityTest:主要演示如何向下个Activity传递数据，以及如何返回数据给前一个Activity
   
3. [相册相机调用](https://blog.csdn.net/weixin_45769562/article/details/155890289?spm=1001.2014.3001.5502)
   主要包括如何调用系统相机，相册，以及如何分享到微信以及其他应用等。
   
5. [Service使用示例](https://blog.csdn.net/weixin_45769562/article/details/155890289?spm=1001.2014.3001.5502)
   演示Service的用法
